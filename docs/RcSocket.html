RcSocket

RcSocket

new RcSocket(url, protocols)

Source:
  • rc-socket.js, line 23
This behaves like a WebSocket in every way, except if it fails to connect, or it gets disconnected, it will use an exponential backoff until it succesfully connects again. It is API compatible with the standard WebSocket API.
Example
const ws = new RcSocket(wss://host)
Parameters:
Name Type Description
url String Url to connect to.
protocols String | Array Optional subprotocols.

reboot()

Source:
  • rc-socket.js, line 132
Refresh the connection if open (close, re-open). If the app suspects the socket is stale (occurs when changing from wifi -> carrier or vice versa), this method will close the existing socket and reconnect.
Example
socket.reboot()
